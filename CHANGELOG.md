# Changelog

Todas las versiones notables de **El Konstructo** se documentan aquí.
Formato basado en [Keep a Changelog](https://keepachangelog.com/es-ES/); versionado
[SemVer](https://semver.org/lang/es/): `vMAYOR.MENOR.PARCHE`.

- **PARCHE** (`1.8.10 → 1.8.11`): corrección de bug, sin cambios visibles de comportamiento.
- **MENOR** (`1.8.10 → 1.9.0`): funcionalidad nueva compatible (p. ej. un universo o modo nuevo).
- **MAYOR** (`1.9.0 → 2.0.0`): cambio grande o incompatible (p. ej. cambia el formato de `quiz_sets`).
